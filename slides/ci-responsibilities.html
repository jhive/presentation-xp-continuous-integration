Team Responsibilities

Check in frequently

Create a habit of smaller commits and integrations: daily, hourly, on the minute

Don't check in broken code

and if you do, everyone stops to fix it

Don't check in untested code

Code that isn't tested, is code that doesn't exist

Don't check in when the build is broken

Even better: Don't write code while the build is broken

Don't go home after checking in until the build is tested

Just wait until you're refreshed in the morning
